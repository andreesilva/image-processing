# API de Processamento de Imagens

Criada em Nestjs

## Endpoints:

- Consulta - http://localhost:3000/api/images/cat.jpg?fm=webp
- Upload de imagem - http://localhost:3000/api/images/upload

- Documentação em : http://localhost:3000/api
